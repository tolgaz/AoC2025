# Advent of Code - Day 6

Go solution for Day 6 of Advent of Code.

## Usage

Run the solution:
```bash
go run main.go
```

Run tests:
```bash
go test
```

Run tests with verbose output:
```bash
go test -v
```

## Files

- `main.go` - Main solution functions and entry point
- `main_test.go` - Test cases
- `go.mod` - Go module definition
- `input.txt` - Puzzle input
- `example.txt` - Example input
